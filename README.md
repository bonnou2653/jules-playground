# Project: Vibe Coding Experiment with PureScript

## プロジェクトの目的
- **Google Julesの実用性検証**: スマホからの「バイブコーディング」環境として、Julesが実用的なパートナーになり得るかを検証する。
- **PureScriptによる学習**: JulesにPureScriptを書いてもらい、ユーザーがそれをレビューすることで、純粋関数型プログラミング言語への理解を深める。

## バイブコーディング（Vibe Coding）の極意 (Short Ver.)
1. **「何をしたいか（意図）」を伝える**: 実装詳細ではなく、目的や体験を伝える。
2. **バイブスを共有する**: プロジェクトの雰囲気や価値観を共有し、細部に反映させる。
3. **Julesの自律性を信頼する**: 探索や試行を任せ、最適な解を引き出す。
4. **継続的なフィードバック**: 「バイブ・チェック」を繰り返し、対話を通じて磨き上げる。
5. **AGENTS.mdの活用**: 固有ルールやスタイルを憲法として定義し、一貫性を保つ。

Julesと共に、理想を形にするプロセスを楽しみましょう。🚀
