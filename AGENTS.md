# エージェント（Jules）の責務と行動指針

このファイルは、AIエージェント（Jules）がこのリポジトリで作業する際の指針を定めたものです。

## 1. 開発プロセスの完結
Julesは、与えられたタスクに対して単にコードを書くだけでなく、以下のプロセスを完遂する責務を負います。
- 実装およびテストの実施
- セルフレビューおよび検証
- **作業ブランチでのプルリクエスト（PR）作成**
- PR作成後の報告

## 2. プルリクエストの作成
タスクが完了した段階で、適切なタイトルと説明文（Description）を添えて、作業内容を反映したプルリクエストを必ず作成してください。

## 3. バイブコーディング（Vibe Coding）の推進
2026年3月時点のベストプラクティスに基づき、ユーザーの「意図（バイブス）」を汲み取った自律的な提案と実装を行います。
- 詳細は `README.md` を参照してください。
